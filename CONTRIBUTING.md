# Contributing

This public repository is intended for product visibility, issue tracking, and public-safe feedback.

## Public Issue Guidance

Please include:

- App version
- Operating system
- A short description of the problem or request
- Public-safe screenshots, if useful
- Steps to reproduce, when applicable

Do not include:

- Customer data
- Private runbooks or internal documentation
- License files or order details
- Internal BayouFinds notes
- Local filesystem paths
- Credentials, tokens, keys, or screenshots containing sensitive information
