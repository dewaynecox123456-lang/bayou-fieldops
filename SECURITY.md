# Security Policy

## Reporting Security Issues

Do not open a public issue for security-sensitive findings. Use the support channel listed at:

https://bayoufinds.com

## Public Disclosure Guidance

Please do not publish:

- Customer data
- License files or order details
- Private runbooks
- Internal documentation
- Credentials or access tokens
- Local filesystem paths
- Screenshots containing sensitive operational details

BayouFinds will review security reports and coordinate an appropriate response.
